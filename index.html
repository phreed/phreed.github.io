Discovering the neural code

The main objective for these techniques was to discover the neural code. There were a number of interesting results from this work. We obtained data from Dr. Steve Potter at Georgia Tech. This data was obtained from a multi-electrode array monitoring embrionic rat brains. The spike trains were prospected for episodes. These episodes are believed to indicate the presence of circuits.

One assumption is that all neural spikes are the same. embrionic rat brain neural waveforms This is clearly not the case.

An indicator that we had actually discovered circuits would be that the shapes of the waveforms that make up these circuits should be visibly similar. The research showed that circuits can be discovered in this way and that the spike patterns match. embrionic rat brain neural episodic waveforms I was supprised by the conformance of these waveforms.

Collaborative engineering – die casting tooling

dieCAS was the first place where I was able to implement collaborative engineering techniques. The technique was fairly primitive, using Sun's ONC/RPC and a central server. It did use speculative locking, rather than preemptive locking. The motivation was due to the participants being distributed geographically. Most other collabortive engineering was performed in close proximity so preemptive locking was practical.

dieCAS is one of a very few software products commercialised from GMR.

I had the opportunity to develop a parametric modeller for die design. I also created parametric models for pistons.
